# 🧩 Pokédex Interativa – Projeto Front-End com PokéAPI

Este repositório apresenta uma Pokédex interativa desenvolvida com HTML, CSS e JavaScript, utilizando a [PokéAPI](https://pokeapi.co/) para exibir informações completas dos Pokémon.

---

## 🎯 Objetivo

Criar uma interface funcional e amigável onde é possível visualizar, pesquisar e explorar dados dos Pokémon diretamente da API.

---

## 🚀 Funcionalidades

- 🔍 Busca por nome ou número do Pokémon  
- 🎨 Exibição visual com nome, imagem, tipo, habilidades e espécie  
- 🌐 Conteúdo traduzido para *português*  
- 📱 Responsivo: adaptado para celular, tablet e desktop  
- 🎮 Design personalizado com toque nostálgico (estilo Pokédex)

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript ES6  
- PokéAPI (API pública)  
- Fontes e ícones personalizados
